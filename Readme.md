# iShadowsocks自动获取密码

### 使用方法

 下载后直接打开批处理文件,点击启用系统代理选项即可。
 
 注意：需要在node.js环境下使用。
 
同时安装有 superagent 和 cheerio 两个npm包。

###ps：
仅支持windows平台
